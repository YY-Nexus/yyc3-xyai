# YYC3 AI小语智能成长守护系统 - 用户使用手册

> 版本: v1.0 | 适用人群: 家长用户

---

## 一、快速入门

### 1.1 注册与登录

1. 打开应用，点击右上角"登录"按钮
2. 选择"注册账号"，填写邮箱和密码
3. 验证邮箱后即可登录使用

### 1.2 添加孩子档案

1. 登录后进入"档案"页面
2. 点击"添加孩子"按钮
3. 填写孩子姓名、昵称、出生日期、性别
4. 点击保存完成创建

### 1.3 切换当前孩子

如果您有多个孩子:

1. 点击页面顶部的孩子头像
2. 在下拉列表中选择要查看的孩子
3. 页面数据自动切换

---

## 二、核心功能

### 2.1 首页概览

首页展示当前孩子的:

- 今日学习计划和作业进度
- 作业中心快捷入口
- 每日一诗文化熏陶
- 个性化问候语

### 2.2 作业管理

**查看作业:**

- 进入"作业"页面查看所有作业
- 使用顶部Tab筛选: 全部/待完成/已完成

**完成作业:**

- 点击作业卡片右侧的勾选按钮
- 状态自动更新为"已完成"

### 2.3 成长记录

**创建记录:**

1. 进入"成长"页面
2. 点击"创建记录"按钮
3. 选择记录类型(里程碑/观察日志/情感记录/学习记录)
4. 填写内容，可上传图片或视频
5. AI会自动分析并生成标签建议

**查看时间线:**

- 在"成长记录"Tab查看所有记录
- 支持时间线/网格两种视图切换
- 按类型筛选特定记录

### 2.4 发展评估

**进行评估:**

1. 进入"成长"页面，切换到"智能评估"Tab
2. 点击"开始评估"按钮
3. 根据孩子实际情况回答问题
4. 完成后查看AI生成的评估报告

**报告解读:**

- 雷达图展示各维度得分
- AI分析提供专业解读
- 建议列表指导下一步行动
- 支持导出PDF报告

### 2.5 AI小语助手

**唤醒方式:**

- 点击右下角悬浮的AI小语图标
- 或使用快捷键 Ctrl+K (Mac: Cmd+K)

**功能面板:**

- 对话: 与AI小语自由交流，获取育儿建议
- 控制: 快捷导航和功能入口
- 情感: 查看孩子情绪分析
- 预测: 智能学习预测和建议
- 设置: AI助手个性化配置

**五大角色:**

- 记录者: 帮助记录成长瞬间
- 守护者: 提供科学育儿指导
- 聆听者: 解读孩子情绪和行为
- 建议者: 提供学习和活动建议
- 国粹导师: 传统文化教育引导

## 三、成长阶段

系统支持0-22岁全周期成长守护:

| 阶段 | 年龄 | 核心关注 |
|------|------|----------|
| 感官启蒙期 | 0-3岁 | 视觉、听觉、运动发展 |
| 游戏学习期 | 3-6岁 | 社交能力、情绪管理 |
| 学术奠基期 | 6-9岁 | 学习习惯、基础学科 |
| 思维建构期 | 9-12岁 | 逻辑思维、创造力 |
| 青春转型期 | 12-15岁 | 心理健康、自我认知 |
| 生涯定位期 | 15-18岁 | 学业规划、职业探索 |
| 成人成才期 | 18-22岁 | 独立发展、社会适应 |

## 五、常见问题

**Q: 如何导出成长报告?**
A: 在评估报告页面点击"导出PDF"按钮

**Q: AI小语可以语音交互吗?**
A: 可以，点击对话输入框旁的麦克风图标开始语音输入

**Q: 数据安全吗?**
A: 所有数据加密存储，严格遵循儿童隐私保护规范

**Q: 可以多人共同管理孩子吗?**
A: 目前支持单账号管理，多人协作功能规划中
