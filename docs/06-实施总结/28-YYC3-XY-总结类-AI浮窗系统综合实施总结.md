## 🚀 部署指南

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/YY-Nexus/yyc3-xy-ai.git
   cd yyc3-xy-ai
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **配置环境变量**
   ```bash
   cp .env.example .env
   # 编辑 .env 文件，配置必要的环境变量
   ```

4. **启动开发服务器**
   ```bash
   npm run dev
   ```

5. **构建生产版本**
   ```bash
   npm run build
   ```

6. **部署到生产环境**
   ```bash
   npm run deploy
   ```

### 配置说明

1. **TypeScript配置**
   - 启用严格模式
   - 配置路径别名
   - 设置编译选项

2. **ESLint配置**
   - 配置代码规范
   - 启用类型检查
   - 设置自动修复

3. **响应式配置**
   - 配置断点系统
   - 设置浮窗尺寸
   - 优化触摸交互

4. **错误处理配置**
   - 配置错误分类
   - 设置恢复策略
   - 配置监控阈值

5. **AI系统配置**
   - 配置学习算法

   - 配置知识库
