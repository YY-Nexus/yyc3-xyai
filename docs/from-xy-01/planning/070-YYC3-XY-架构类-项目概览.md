# YYC³ AI小语项目概览

**项目全称**: YYC³ AI小语智能成长守护系统
**项目代号**: xy-ai
**当前版本**: v1.0 Birthday Edition
**项目状态**: 🚀 开发中 (生日版本冲刺)

---

## 💝 项目使命

为0-22岁全生命周期的孩子提供智能化的成长守护，用AI技术记录每一个珍贵瞬间，用温暖的陪伴守护成长之路。

**核心理念**: 智能记录 + 科学指导 + 情感陪伴

---

## 🏗️ 项目架构总览

### 📊 项目规模
- **总文件数**: 350个文件
- **代码行数**: 约50,000行
- **组件数量**: 85个React组件
- **页面数量**: 26个功能页面
- **微服务**: 15个后端服务

### 🛠️ 技术栈全景
```
🎨 前端技术栈
├── Next.js 14.2.33 (React框架)
├── TypeScript 5 (类型安全)
├── TailwindCSS 4.1.9 (样式框架)
├── Framer Motion (动画库)
├── Radix UI (组件库)
└── Lucide React (图标库)

⚡ 运行时环境
├── Bun 1.1.38 (高性能运行时)
├── Node.js 18+ (兼容性支持)
└── React 18 (UI框架)

🧠 AI与后端
├── OpenAI API (大语言模型)
├── Azure Speech (语音服务)
├── PostgreSQL (主数据库)
├── Redis (缓存系统)
└── Socket.io (实时通信)
```

### 🎯 核心功能模块
```
🧠 AI智能体系统 (5种智能体)
├── 记录者 - 时光存档专家
├── 守护者 - 安全保护专家
├── 倾听者 - 情感理解专家
├── 顾问 - 学习指导专家
└── 文化导师 - 传统文化专家

📱 主要功能页面
├── 成长追踪 (/growth) - 记录成长轨迹
├── AI对话 (/ai-chat) - 智能互动交流
├── 作业助手 (/homework) - 学习辅导支持
├── 课程推荐 (/courses) - 个性化学习
├── 情感监控 (/emotion) - 情绪状态分析
├── 图书阅读 (/books) - 阅读习惯培养
├── 视频生成 (/videos) - 创意内容制作
└── 家长中心 (/settings) - 管理控制面板
```
